# 서버 실행
- 파워쉘 / 커맨드 / 터미널 창 오픈
- cd ./be/자신의 기능 폴더명
- npm run dev 입력

# 서버 재시작
- 파일을 변경 한 뒤
- 저장(Control + S)

# 서버 종료
- Control + C

# GitHub에 변경사항 저장하기
- 파일을 전부 저장
- 파워쉘 / 커맨드 / 터미널 창 오픈
- [중요] 자신의 현재 위치가 모든 폴더의 최상위 위치(linux-project)인지 확인
- git add .
- git commit -m "변경사항 제목"
- git push origin 자신의 브랜치 이름
